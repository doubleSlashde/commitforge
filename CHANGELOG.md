# Changelog

## [1.0.0] - 2026-03-24

### Features
- One-click commit message generation via Claude Code CLI
- Conventional Commits, descriptive, and custom formats
- Staged-first diff detection (falls back to all changes)
- Per-project format configuration via `.vscode/settings.json`
- Configurable language (English / German)
- Configurable max diff size
- Clean output: no trailers, no co-authored-by, no code fences
